<h3>1. <strong>Python 3.x</strong>:</h3>
<ul>
  <li>Ensure you have <strong>Python 3.x</strong> installed.</li>
  <li><a href="https://www.python.org/downloads/" target="_blank">Download Python</a>.</li>
</ul>

<h3>2. <strong>PyTorch</strong>:</h3>
<ul>
  <li>Install PyTorch and torchvision:
    <pre><code>pip install torch torchvision torchaudio</code></pre>
  </li>
  <li>For CUDA-enabled PyTorch (with GPU support), use:
    <pre><code>pip install torch torchvision torchaudio cudatoolkit=11.3</code></pre>
  </li>
</ul>

<h3>3. <strong>NumPy</strong>:</h3>
<ul>
  <li>For numerical computations:
    <pre><code>pip install numpy</code></pre>
  </li>
</ul>

<h3>4. <strong>Pandas</strong>:</h3>
<ul>
  <li>For data manipulation and analysis:
    <pre><code>pip install pandas</code></pre>
  </li>
</ul>

<h3>5. <strong>Matplotlib</strong> and <strong>Seaborn</strong>:</h3>
<ul>
  <li>For data visualization:
    <pre><code>pip install matplotlib seaborn</code></pre>
  </li>
</ul>

<h3>6. <strong>OpenCV</strong>:</h3>
<ul>
  <li>For image processing tasks:
    <pre><code>pip install opencv-python</code></pre>
  </li>
</ul>

<h3>7. <strong>scikit-learn</strong>:</h3>
<ul>
  <li>For machine learning utilities and evaluation:
    <pre><code>pip install scikit-learn</code></pre>
  </li>
</ul>

<h3>8. <strong>TensorBoard</strong> (Optional):</h3>
<ul>
  <li>For monitoring model training:
    <pre><code>pip install tensorboard</code></pre>
  </li>
</ul>

<h3>9. <strong>Jupyter Notebook</strong> (Optional):</h3>
<ul>
  <li>For interactive development:
    <pre><code>pip install jupyterlab</code></pre>
  </li>
</ul>

<h3>10. <strong>Git</strong>:</h3>
<ul>
  <li>Install Git for version control:
    <ul>
      <li><a href="https://git-scm.com/" target="_blank">Download Git</a></li>
    </ul>
  </li>
</ul>

<h3>11. <strong>CUDA Toolkit</strong> (Optional for GPU):</h3>
<ul>
  <li>If you want to use GPU acceleration, install the CUDA toolkit. Ensure compatibility with PyTorch version:
    <ul>
      <li><a href="https://developer.nvidia.com/cuda-toolkit" target="_blank">Download CUDA Toolkit</a></li>
    </ul>
  </li>
</ul>

<h3>12. <strong>Pillow</strong>:</h3>
<ul>
  <li>For image file handling and processing:
    <pre><code>pip install pillow</code></pre>
  </li>
</ul>

<h3>13. <strong>Setting up a Virtual Environment</strong> (Recommended):</h3>
<ul>
  <li>Use virtual environments to isolate your project dependencies:
    <pre><code>python -m venv solar-panel-env</code></pre>
    <pre><code>source solar-panel-env/bin/activate  # On Windows: solar-panel-env\Scripts\activate</code></pre>
  </li>
</ul>
